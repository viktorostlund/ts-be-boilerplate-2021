# Typescript backend boilerplate 2021

Basic NodeJS/Typescript API for getting or creating data about people to and from a database.

## Endpoints

- createPerson
- getPerson
## Run the app

Run app: `npm run dev`

Compile TS and then run app: `npm start`
## Technologies
- GraphQL
- Postgres
- Express
